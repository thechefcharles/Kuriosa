## Notes

- We intentionally include **more than 25** so you can skip a few if any generator output fails validation.
- Category labels should match (or be mappable to) your seeded `categories` table. Current DB seed includes: `Science`, `History`, `Nature`, `Technology`, `Culture`. If you want `Psychology` / `Space` / `Finance/Economics` / `Philosophy` as categories, add them to `categories` first (Phase 4.11 will not redesign schema, but adding reference categories is safe).
