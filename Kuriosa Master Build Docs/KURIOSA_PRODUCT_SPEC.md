# Kuriosa — Product Specification

## Overview

Kuriosa is a curiosity-driven learning app designed to help people discover fascinating knowledge in short, engaging daily experiences.

Instead of traditional courses or endless scrolling content, Kuriosa delivers small curiosity adventures that take only 5–10 minutes to complete.

Users open the app, discover something surprising, learn the concept, test themselves with a challenge, and continue exploring related ideas.

The product blends elements of:
- Wikipedia rabbit holes
- Duolingo-style gamification
- Wordle-style daily engagement

**The goal is to make curiosity addictive.**

---

## Core Concept

Kuriosa is built around a simple daily loop:

```
Open the app
    ↓
Discover something fascinating
    ↓
Read or listen to a short explanation
    ↓
Answer a challenge question
    ↓
Earn XP and progress
    ↓
Follow curiosity trails into related topics
```

Each session is designed to take 3–5 minutes.

The experience is lightweight, educational, and satisfying.

---

## Example Experience

**Topic:** Why do octopuses have three hearts?

**User flow:**
1. A curiosity card appears.
2. The user reads or listens to a short explanation.
3. A quiz challenge appears.
4. The user earns XP.
5. Suggested curiosity trails appear.

**Example curiosity trail:**
- Octopus Hearts → Cephalopod Intelligence → How Camouflage Works → Color Changing Skin

## Key Product Features

### Daily Curiosity
Each day users receive a new curiosity topic. This creates a daily learning habit.

### Random Discovery — "Feed My Curiosity"
Signature feature: Users tap a button to spin a random topic. Randomness keeps discovery exciting.

### Curiosity Lessons
Each topic contains:
- hook question
- 200–400 word explanation
- surprising fact
- real-world relevance
- optional audio narration

Lessons are short, engaging, and easy to understand.

### Quiz Challenges
Every topic includes a challenge. Examples:
- multiple choice
- reasoning questions
- logic questions

Challenges reinforce learning.

### Curiosity Trails
After completing a topic, users receive related topics to explore. This creates a knowledge rabbit hole effect.

### Audio Learning Mode
Users can listen to lessons instead of reading. Perfect for commuting, walking, working out. Audio turns Kuriosa into a micro podcast-style learning tool.

### Gamification
Users earn:
- XP
- levels
- streaks
- badges
- Curiosity Score

These systems reward exploration.

### Follow-Up Questions
Users can ask curiosity questions about the topic. AI provides a short explanation. This deepens curiosity without turning the product into a chatbot.

### Personalization
As users explore topics, Kuriosa learns their interests. The system adapts to recommend similar topics, deeper trails, and personalized daily curiosities.

**The goal:** "Kuriosa knows what I'm curious about."

## Content Model

Each curiosity topic contains structured content:

- title
- hook question
- lesson text
- surprising fact
- real-world relevance
- quiz challenge
- follow-up questions
- category
- difficulty
- tags
- optional audio narration

Content is generated and expanded using AI but served from a stored content library. This ensures reliability, moderation, and fast loading.
